# Selenium Contact Form Automation

## 📌 Project Overview
This project automates the "Contact Us" form using Selenium WebDriver in Java.  
It performs both positive and negative test scenarios including validation checks.

---

## 🚀 Features

### ✔ Positive Flow
- Navigate from Home page to Contact page
- Fill contact form fields
- Submit form
- Validation of the form

### ✔ Negative Flow
- Submit empty form
- Validate required field errors or HTML5 validation

### ✔ Extra Features
- reCAPTCHA test mode support
- Screenshot capture on failure
- Multiple click strategies (Actions + JS Executor)
- Robust waits using WebDriverWait

---

## 🔧 Technologies Used
- Java
- Selenium WebDriver
- ChromeDriver
- WebDriverWait (Explicit Waits)
